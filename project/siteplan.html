W05 Site Plan

Stories and Legends of Huancavelica

1. Naming Conventions

All files and folders will follow the course naming conventions using lowercase words separated by hyphens.

  • index.html
  • stories.html
  • story-page.html
  • styles.css
  • images/, scripts/

2. HTML Validation

All HTML documents for this project will be validated using the W3C HTML Validator. I will make sure that each page reports no errors.

3. CSS Validation

All CSS will be validated using the W3C CSS Validator to ensure there are no errors in the stylesheet.

4. Site Name

Stories and Legends of Huancavelica

This name reflects the main purpose of the website: to share traditional stories, myths, and legends from the Huancavelica region in an educational and accessible way.

5. Site Purpose

The purpose of this website is to help students improve their reading skills through culturally meaningful content while preserving Huancavelica’s rich storytelling traditions. The site will provide teachers with easy-to-access reading materials and present a curated collection of local legends in a clear, modern web format.

6. Scenarios

  1. Student Scenario: A middle school student needs an engaging story to practice reading comprehension for homework. They visit the website on their phone, open the Stories page, and choose a legend to read.
  2. Teacher Scenario: A teacher preparing a reading activity wants culturally relevant material for her students. She accesses the site from her classroom computer, reviews several stories, and selects one to use in a lesson.

8. Color Schema Used on Document

This site plan and the final website will apply the selected colors consistently:

  • Navigation bar and headings: Deep Andean Blue
  • Buttons and interactive elements: Warm Andean Red
  • Main reading sections: Cultural Beige
  • Body background and text areas: White

9. Typography

The website will use Google Fonts:

  • Montserrat – for titles, headings, and navigation items.
  • Lora – for body text and story content.

10. Typography Used on Document

This site plan document uses the same fonts:

  • Section headings are displayed in Montserrat.
  • Body text is displayed in Lora.
